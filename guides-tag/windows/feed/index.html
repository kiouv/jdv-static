]]> Star Citizen ne permet pas aux joueurs de savoir combien de temps ils passent dans le Verse. Pour certains, c’est peut-être une bonne chose…

​Le projet scplay est un calculateur de temps de jeu pour Star Citizen, développé en C# (précédemment en Python). Il analyse les fichiers journaux du jeu (*.log) situés dans le dossier « logbackups » de votre installation Star Citizen pour calculer votre temps total de jeu.

Note: Le Journal du Verse rappelle qu’il est toujours important de faire preuve de prudence lorsqu’on télécharge et exécute des fichiers depuis Internet. Avant d’utiliser un programme tiers, assurez-vous qu’il provient d’une source fiable et, si possible, analysez-le avec un antivirus. La sécurité avant tout, même dans le Verse !

SCPlay sur Github

Fonctionnalités principales :

  • Parcourt les fichiers journaux pour identifier les dates et heures à l’aide d’expressions régulières.​
  • Calcule les différences de temps au sein de chaque fichier, puis additionne ces durées pour obtenir le temps total de jeu.​

Utilisation :

  1. Téléchargez l’exécutable disponible dans la section des releases du dépôt.​
  2. Ouvrez l’application et sélectionnez le dossier « logbackups » de votre installation Star Citizen.​
    • Par défaut, ce dossier se trouve généralement à l’emplacement suivant : C:\Program Files\Roberts Space Industries\StarCitizen\LIVE\logbackups.​
    • Remarque : « LIVE » correspond à l’environnement de production standard ; si vous utilisez d’autres environnements, sélectionnez le dossier approprié.​

Si vous préférez, vous pouvez également cloner le dépôt et compiler la solution vous-même en utilisant Visual Studio 2022 (édition Community). Le projet est sous licence MIT, vous êtes donc libre de l’utiliser et de le modifier selon vos besoins.
